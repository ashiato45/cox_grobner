Calculates groebner basis of 
\begin{itemize}
\item $x+(-1)z^{4}$
\item $y+(-1)z^{5}$
\end{itemize}  .


$\overline{S(x+(-1)z^{4}, y+(-1)z^{5})} = 0$.

Enough for groebner basis.  Result is \begin{itemize}
\item $x+(-1)z^{4}$
\item $y+(-1)z^{5}$
\end{itemize}  .
$\blacksquare{}$
Minimalizes groebner basis
\begin{itemize}
\item $x+(-1)z^{4}$
\item $y+(-1)z^{5}$
\end{itemize}  .


Minimalized groebner basis is
\begin{itemize}
\item $x+(-1)z^{4}$
\item $y+(-1)z^{5}$
\end{itemize}  .
$\blacksquare{}$

Reduce groebner basis
\begin{itemize}
\item $x+(-1)z^{4}$
\item $y+(-1)z^{5}$
\end{itemize}  .


Reducing: $\overline{x+(-1)z^{4}} = x+(-1)z^{4}$.

Reducing: $\overline{y+(-1)z^{5}} = y+(-1)z^{5}$.

Reduced groebner basis is
\begin{itemize}
\item $y+(-1)z^{5}$
\item $x+(-1)z^{4}$
\end{itemize}  .
$\blacksquare{}$
